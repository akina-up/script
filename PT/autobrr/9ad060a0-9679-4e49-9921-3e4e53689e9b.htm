HDBits provides a JSON API which will allow you to get useful data from the site. All methods exist at https://hdbits.org/api/:method

Note: There is a limit of 35 downloaded .torrent files in a 15 minute rolling window. Further requests are rejected with a 403 error. Make sure your scripts use appropriate delays.

Authentication / Minimum request

Every request requires authentication by username and passkey, or username and password.

API doesn't use 2FA. But from 2023-06-22 API usage is allowed only for users with active 2FA for the site.

Request Params

Field Description
username
passkey Optional The same passkey as in your tracker announce url
password Optional Your account password

Response

Field Type Description
status int A code indicating success (0) or failure (any other number). See status codes
message string Optional A message explaining the error.
data object Optional If the request is successful the method will put any data requested in this field


Example

Request: 

POST https://hdbits.org/api/test
{
  "username": "aa",
  "passkey": "bb"
}

Response: 

{
  "status":5,
  "message":"Invalid authentication credentials"
}

Wishlist

https://hdbits.org/api/wishlist

Returns the user's wishlist

Request Params

None

Response

An array of wishlist items

Field Description
id Wishlist item ID
name Display name of the item
imdb IMDb ID of the item (number)
added Date added to wishlist (yyyy-mm-dd)
originaltitle The original title of the item (from IMDb)
englishtitle The english title of the item (from IMDb)
year The IMDb year of the item
torrent Is there a torrent with this imdb number?

Example

 POST https://hdbits.org/api/wishlist
 {"username":"aa","passkey":"bb"}

 }

WishlistAdd

https://hdbits.org/api/wishlistAdd

Add a film to your wishlist

Request Params

Field Type Description
imdb_id int IMDb ID of the film you want to add. e.g. The ID of the matrix http://www.imdb.com/title/tt0133093/ is 133093

Response

There is no response data. If the status code is 0 the film has been added to your wishlist.

Possible status codes

0-9. See status codes

WishlistDel

https://hdbits.org/api/wishlistDel

Remove an item from your wishlist

Request Params

Field Type Description
id int ID of the item you want to remove from your wishlist

Response

There is no response data. If the status code is 0 the item has been removed from your wishlist.

Possible status codes

0-9. See status codes


Torrents

https://hdbits.org/api/torrents

Retrieves up to 30 torrents filtered by any request params. By default this will just be the latest 30 torrents.

Request Parameters

All parameters are optional

Field Type Description
id int ID of a torrent
hash string 40 character hex string representing the info hash of the torrent e.g. "E94DB7477A9037748782C7F311043C7E9D2FF711"
search string Search terms to look for
category int[] Only return torrents in these categories (1 Movie, 2 TV, 3 Documentary, 4 Music, 5 Sport, 6 Audio Track, 7 XXX, 8 Misc/Demo)
codec int[] Only return torrents with these codecs (1 H.264, 2 MPEG-2, 3 VC-1, 4 XviD, 5 HEVC)
medium int[] Only return torrents in these mediums (1 Blu-ray/HD DVD , 3 Encode, 4 Capture, 5 Remux, 6 WEB-DL)
origin int[] Only return torrents with these origins (0 Undefined, 1 Internal)
exclusive int[] Only return torrents with these exclusivities (0 Non-exclusive, 1 Exclusive)
tags int[] Only return torrents that have one of these tags
imdb imdb Supports id lookups
tvdb tvdb Supports id, episode and season lookups (ID required, others are optional)
file_in_torrent string Name of a file in the torrent
snatched_only boolean If true only include torrents this user has been on
limit int The maximum number of torrents to retrieve (1-100). Default is 30
page int The page of torrents to retrieve. e.g. To get the 30th to 60th torrent set page to 1. Default is 0

Response

An array of torrent items:

Field Type Description
id int Torrent ID
hash string 40 character hex string representing the info hash of the torrent e.g. "E94DB7477A9037748782C7F311043C7E9D2FF711"
leechers int Number of leechers on this torrent
seeders int Number of seeders on this torrent
name string
descr string Description in BBCode format
times_completed int
size int Size of the files in the torrent in bytes
utadded int Unix timestamp of when the torrent was uploaded
added datetime Timestamp of when the torrent was uploaded
comments int Number of comments on this torrent
numfiles int Number of files in this torrent
filename string Original filename of the torrent file
freeleech string "yes" or "no"
type_category int Category ID
type_codec int Codec ID
type_medium int Medium ID
type_origin int Origin ID
type_exclusive int Exclusivity ID
torrent_status string User's torrent status - "", "seeding", "leching" or "completed"
bookmarked int 1 or 0
wishlisted int 1 or 0
tags string[] Array of tag names associated with this torrent
imdb imdb An object representing the IMDb data for this film. Field only exists if imdb data exists
tvdb tvdb A tvdb object. Field only exists if tvdb data exists

Top 50 films (by downloads)

https://hdbits.org/api/topFilm

Returns the top 50 films for certain time periods

Request Params

Field Type Description
list string Which list you want. One of: FilmNewThisWeek, FilmNewThisMonth, FilmNewLastMonth, FilmNewThisYear

Response

An array of items

Field type Description
imdb_id string IMDb ID padded with 0s to 7 digits
name object Object containing 'original' and 'english'
amount int Amount of times downloaded

Example

Request: 

 POST https://hdbits.org/api/topFilm
 {"username":"valerio","passkey":"xxx", "list": "FilmNewThisMonth"}

       "banner":1,
       "imdb_id":"2294629",
       "name":{
         "original":"Frozen",
         "english":"Frozen"
       },
       "amount":5217
     },
     {
       "banner":1,
       "imdb_id":"0816442",
       "name":{
         "original":"The Book Thief",
         "english":"The Book Thief"
       },
       "amount":1113
     }
   ]
 }

Subtitles

https://hdbits.org/api/subtitles

Returns up to 100 subtitles as filtered by the optional request parameters. You can download the subtitle at /getdox.php?id=***&passkey=***

Request Params

Field Type Description
torrent_id int The ID of the torrent the subtitle is directly linked to. All subtitles are linked to one torrent.

Response

An array of subtitles

Field type Description
id int
added datetime When this subtitle was uploaded
title string
filename string The original filename for the subtitle
language language
torrent_id int ID of the torrent this subtitle is linked to

Rss Add

https://hdbits.org/api/rssAdd

Add a torrent to your personal rss feed

Request Params

Field Type Description
torrent_id int ID of the torrent you want to add

Response

There is no response data. If the status code is 0 the torrent has been added.

Possible status codes

0-7. See status codes

Types

IMDB

Field Type Description
id int IMDb ID of this film
englishtitle string English title of this film
originaltitle string Original title of this film
year int IMDb year of this film
genres string[] array of genres for this film e.g. ["Animation","Drama"]
rating float IMDb rating of this film

TVDB

Field Type Description
id int TVDB ID of this show
season int Season for this torrent. Specials are season 0 (see thetvdb.com)
episode int Episode for this torrent. Season packs are episode 0.

Datetime

ISO-8601 formatted representation of time. e.g. 2005-08-15T15:52:01+0000

Status Codes

Each response has a field called status indicating the success or failure of the request. 0, 1, 3, 4 and 5 are general errors and authentication errors and can be returned by any method.

Code Description
0 Success
1 Failure (something bad happened)
2 SSL Required - You should be on https://
3 JSON malformed
4 Auth data missing
5 Auth failed (incorrect username / password)
6 Missing required parameters
7 Invalid parameter
8 IMDb - Import fail
9 IMDb - TV not allowed

API for Imagehost

We provide POST API for uploading images to our imagehost.

URL of the API is: https://img.hdbits.org/upload_api.php

Maximum image size is 50 MiB. Maximum POST size is 500 MiB.

Request Params

Field Description
username Required if not authorized for HDBits with cookies
passkey Required if not authorized for HDBits with cookies
thumbsize Optional Thumbnail size, default = 'w300' (also follows user's saved defaults)

other options ?100/150/200/250/300/350 where ? can be
w = Resize, Keep Aspect Ratio (Width)
h = Resize, Keep Aspect Ratio (Height)
d = Resize, Keep Aspect Ratio (Largest Dimension)
s = Resize & Crop (Square)

galleryoption Optional Gallery options

default = '1' = Create Gallery
'0' = Do Not Create Gallery
'2' = Upload to Existing Gallery

galleryname Optional Name of the gallery (used for galleryoption = '1', default = 'Untitled Gallery')
existgallery Optional ID of existing gallery (owned by same user) (required for galleryoption = '2')
images_files[] Array of one or more images, sent as multipart/form-data

Response

BBCode as plain text (or error starting with 'error:')

Example with curl and PHP

function sendPOST($url, $postfields, $files) {
  $content = false;
  $curl = curl_init();
  curl_setopt($curl, CURLOPT_URL, $url);
  curl_setopt($curl, CURLOPT_POST, true);
  if (isset($files)) {
    foreach ($files as $id=>$file) {
      $postfields["images_files[$id]"] = $file;
    }
  }
  curl_setopt($curl, CURLOPT_POSTFIELDS, $postfields);
  curl_setopt($curl, CURLOPT_HTTPHEADER, array('Content-Type: multipart/form-data'));
  curl_setopt($curl, CURLOPT_RETURNTRANSFER, true);
  curl_setopt($curl, CURLOPT_FRESH_CONNECT, true);
  curl_setopt($curl, CURLOPT_IPRESOLVE, CURL_IPRESOLVE_V4);
  curl_setopt($curl, CURLOPT_TIMEOUT, 600);
  curl_setopt($curl, CURLOPT_CONNECTTIMEOUT, 600);
  
  $content = curl_exec($curl);
  curl_close($curl);
  return $content;
}

$url = 'https://img.hdbits.org/upload_api.php';
$postfields = array();
$postfields['username'] = 'username';
$postfields['passkey'] = 'passkey';
$postfields['thumbsize'] = 'w300';
$postfields['galleryoption'] = '1';
$postfields['galleryname'] = 'My API Gallery';

$files = array();
$files[0] = new CurlFile("img01.png");
$files[1] = new CurlFile("img02.png");
$files[2] = new CurlFile("img03.png");

$bbcode = sendPOST($url, $postfields, $files);

echo $bbcode;
exit();
